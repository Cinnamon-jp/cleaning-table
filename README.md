# cleaning-table

## Excelの文法
- `#` を1列目に書いた行はコメント行として扱う
- コメント行を除いた一番上の行に部屋番号を記載する。
  - 部屋番号は `,` で区切って表記し、指定する部屋番号が続く場合は `:` を用いて範囲指定できる。
  - 例: `101, 103, 105:107` = `101, 103, 105, 106, 107`
- 部屋番号指定行の下にランダム割り当てする役職を、`*` を用いて数とともに記載する。
  - 例: `フロア*3` `洗濯室*2` = `フロア3人` `洗濯室2人`
  - `?` を用いて、その列に割り当てられた部屋番号の残数をその役職に割り振ることができる。*同じ列で一度しか使えない。*
  - `トイレ*?` = `トイレ*まだ割り当てられてない部屋番号総数`
- ランダム割り当ては列単位で行われる。
