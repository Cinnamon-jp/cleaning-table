# プロジェクト概要
- 本プロジェクトは学生寮の毎週の掃除当番をエクセルファイルを基にランダムに生成し、掃除当番表として印刷できるPDFファイルに出力するCLIツールです。
- 本プロジェクトは *Go v1.26.2* を開発言語とします。
- 各種リンタ・フォーマッタには *golangci-lint v2.11.4* を使用します。
- 外部ライブラリとして、エクセルの読み込みには *Excelize v2.10.1* 、PDFの書き出しには *Maroto v2.4.0* を使用します。
- 全ての種類のログには `log/slog` を使用します。

# ルール
- このファイルが存在するディレクトリより上の階層のディレクトリに影響する操作は、絶対に行わないでください。
- git, jjなどを用いたバージョン管理操作は、絶対に行わないでください。
- 関数を作成した場合は必ず `_test.go` ファイルにテストを作成しなさい。
