# 仕様
- `<room_number>~<room_number>` で部屋番範囲を指定(`,` による区切り可能)。
- `<task>*<number>` で役職と数を指定(`<number>`を`?`にすることで、未割り当ての部屋の残数を代入する)。
- `?`は各部屋番範囲に1つのみ。
- 部屋番範囲が重複する場合はエラーを出す。
- 部屋数と役職数が異なる場合はエラーを出す。
- 同じ部屋に複数の役職が割り当てられる場合はエラーを出す。
- 実行ファイルと同じ階層に複数`.xlsx`ファイルが存在する場合は選択できるUIを表示する。
- 役職に`*`が指定されていない場合は(2個以上指定されている場合も)エラーを出す。
