# プロジェクト概要
- 本プロジェクトは学生寮の毎週の掃除当番をエクセルファイルを基にランダムに生成し、掃除当番表として印刷できるPDFファイルに出力するCLIツールです。

# プロジェクト構成
- `main.go`: プログラムのエントリポイント。
- `excel.go`: エクセルを読み込み、エクセルの文法チェックを行う。
- `shuffle.go`: チェックされたエクセルのデータ(スライス)を受け取り、ルールに沿って掃除当番を各部屋に割り振る。
- `pdf.go`: シャッフルされた結果を受け取り、掃除当番表のPDFファイルを作成する。1ページごとに1階分記載し、合計9ページ作成する。
  **詳細**
  - 縦長のA4サイズで作成する。
  - ページのタイトル(上部中央)に、**1F掃除当番表**のように階数と「掃除当番表」を表示する。
  - 表は01号室から29号室までと、30号室から49号室までの2つに分ける。
  - 二つの表の間には適切な間隔を設ける。
  - 以下に例を示す。

  | 部屋番号 | 掃除場所 |  | 部屋番号 | 掃除場所 |
  | --- | --- |         | --- | --- |
  | 01 | フロア |        | 30 | 洗濯室 |
  | 02 | トイレ |        | 31 | ゴミ分別 |
  | ... | ... |         | ... | ... |
  | 29 | フロア |        | 49 | ゴミ分別 |
- `*_test.go`: 各ファイルにおけるテストファイル。
- `test.xlsx`: テスト用のエクセルファイル。正確な文法で記述されている。

# プロジェクトルール
- 各 `.go` ファイルのデータを他の `.go` ファイルに送信するときは、 `main.go` ファイルを通して通信すること。
- 全てのログ(Info, Warn, Error)の本文には、英語の文と日本語の文を両方記載すること。

# エクセル
## 基本文法
- 1行目で抽選対象の部屋番号を指定する。一部屋ずつ `,` で区切って指定するが、指定が続く範囲は `:` を使って省略できる。例: `[101, 103:105, 107]` = `[101, 103, 104, 105, 107]`
- 2行目以降は同列一行目の部屋番号範囲に割り振っていく役職を `役職名*役職数` の形式で指定していく。例: `ゴミ分別*3` = ゴミ分別3人, `トイレ*2` = トイレ2人, etc.。なお、各列に一度だけ `?` を使い、残っている部屋の数だけ自動で割り振るようにする。例: `フロア*?` = フロア(一行目の部屋番号総数-他の役職の人数の合計)人。
- 1列目に `#` だけが記述されたセルが存在する行はコメントとして扱う。

## 文法詳細ルール
- 部屋番号は学生寮に存在する部屋だけ指定可能。(この学生寮に存在する部屋は **1F~9F** まで各階 **01~49号室**。つまり、 `101, 102, 103, ... , 149, 201, 202, ... , 949`)
- スペースは基本的に無視するが、数字と数字の間に入っていた場合はエラーとする。例: `101 -> OK`, `10 1 -> ERROR`。
- 割り当て数自動決定用の `?` は各列で1度だけ使用可能。(割り当て数が決まらないことを防ぐため。)
- 部屋番号の重複指定、役職の重複割り当てが発生した場合はエラーとする。
- 部屋番号総数と役職総数が一致しない場合はエラーとする。( `?` が指定されていない場合)
- 不正文法によるエラー発生時には、エラーメッセージにエラーが発生したセルを明記する。例: `A1`, `B4`
